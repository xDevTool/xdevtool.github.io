# DevKit Legal Documents

This repository contains the privacy policy and terms of service for DevKit Android application.

## 🌐 Supported Languages

Legal documents are available in:

1. **🇺🇸 English** - For international users
2. **🇨🇳 简体中文 (Simplified Chinese)** - For Chinese users

> **Note:** We focus on these two languages as they cover the majority of Android developers worldwide. English serves as the international standard, while Simplified Chinese covers the largest Android market.

## Files

### Main Pages
- `index.html` - Main landing page with links to legal documents
- `privacy.html` - Privacy Policy page with language switcher
- `terms.html` - Terms of Service page with language switcher

### Content Files
- `privacy-content.js` - Privacy Policy content (English + Simplified Chinese)
- `terms-content.js` - Terms of Service content (English + Simplified Chinese)

### Styling
- `style.css` - Shared stylesheet for all pages

## Features

- **Single Page Multi-Language**: Each document (Privacy Policy and Terms of Service) is a single HTML page with a language switcher
- **Dynamic Content Loading**: Content is loaded dynamically based on selected language
- **Language Persistence**: User's language preference is saved in localStorage
- **Responsive Design**: Works seamlessly on desktop, tablet, and mobile devices
- **Clean Modern UI**: Professional gradient design with smooth transitions

## Key Points

### Privacy Policy
- **No Data Collection**: DevKit does not collect or transmit any personal data to external servers
- **Local Storage Only**: All data is stored locally on the user's device
- **No Tracking**: No analytics, advertising, or tracking services
- **Transparent Permissions**: Clear explanation of all requested permissions

### Terms of Service
- **Open Source**: DevKit is open-source software
- **Free to Use**: Provided free of charge for developers
- **Acceptable Use**: Guidelines for lawful and ethical use
- **No Warranties**: Provided "as is" without guarantees

## Deployment

These files are designed to be hosted on GitHub Pages at `xdevtool.github.io`.

To deploy:
1. Push this directory to the `xdevtool.github.io` repository
2. Enable GitHub Pages in repository settings
3. Access at `https://xdevtool.github.io/`

## Updating

When updating legal documents:
1. Update the "Last Updated" date in each file
2. Make changes consistently across all language versions
3. Notify users of significant changes through app updates
