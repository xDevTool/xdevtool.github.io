Android Developer Toolbox

一款为 Android 开发者打造的实用工具箱应用，集成了应用管理、设备信息、统计分析和开发者工具等功能。

核心功能

强大的功能集，助力 Android 开发

📱

应用管理

  • 浏览已安装应用
  • 筛选用户/系统/可调试应用
  • 查看应用详情（权限、组件、证书）
  • 查看 Manifest 文件
  • 导出 APK 文件
  • 检测应用框架（Flutter、React Native、Unity 等）
📊

设备信息

  • 系统信息（设备名称、品牌、Android 版本）
  • CPU 信息（处理器、核心数、架构）
  • 内存使用情况
  • 存储空间
  • 屏幕信息（分辨率、密度、刷新率）
  • 网络状态
📈

统计分析

  • 应用总览统计
  • 应用类型分布图
  • 应用大小分布
  • 安装时间线
  • Target SDK / Min SDK 分布
  • 框架类型统计
🛠️

开发者工具

  • 当前 Activity 悬浮窗
  • 屏幕取色器
  • 二维码扫描与生成
  • DP/PX 转换器
  • 文本编码工具
  • JSON 格式化
  • 屏幕标尺
  • Intent 发送器
