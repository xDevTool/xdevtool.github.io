# DevKit Legal Documents - Summary

## 📋 Overview

This repository contains bilingual legal documents for the DevKit Android application:
- **Privacy Policy** - Explains data handling and privacy practices
- **Terms of Service** - Defines usage terms and conditions

## 🌐 Languages

- 🇺🇸 **English** - International standard
- 🇨🇳 **简体中文** - For Chinese market

## 🔗 Quick Links

- **Main Page:** [index.html](https://xdevtool.github.io/)
- **Privacy Policy:** [privacy.html](https://xdevtool.github.io/privacy.html)
- **Terms of Service:** [terms.html](https://xdevtool.github.io/terms.html)

## 🎯 Key Features

### Privacy Policy Highlights
✅ **100% Local Storage** - All data stays on your device  
✅ **Zero Data Collection** - No personal data transmitted  
✅ **No Tracking** - No analytics, ads, or cookies  
✅ **Transparent Permissions** - Clear explanation of all permissions  

### Terms of Service Highlights
✅ **Open Source** - Free and open-source software  
✅ **Developer-Friendly** - Made for Android developers  
✅ **Clear Guidelines** - Acceptable use policies  
✅ **No Warranties** - Provided "as is"

### Language Detection

The pages automatically remember user's language preference using localStorage. Users can switch languages using the dropdown at the top of each page.

## 🛠️ Technical Details

- **Framework:** Vanilla HTML/CSS/JavaScript
- **Hosting:** GitHub Pages
- **Responsive:** Mobile, tablet, and desktop friendly
- **Performance:** Fast loading with minimal dependencies

## 📝 Maintenance

When updating legal documents:

1. Edit `privacy-content.js` for Privacy Policy changes
2. Edit `terms-content.js` for Terms of Service changes
3. Update the "Last Updated" date in both languages
4. Test both English and Chinese versions
5. Commit and push to GitHub
